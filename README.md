
# 📊 CodeAlpha Data Visualization - Quotes Dataset

## 📌 Project Overview
This project focuses on data visualization using Python. The dataset contains famous quotes, authors, and tags. Various charts were created to identify patterns, trends, and insights from the data.

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📂 Dataset
The dataset contains:
- Quote
- Author
- Tags

## 📈 Visualizations Created

### 1. Top 10 Authors by Number of Quotes
Shows the authors with the highest number of quotes in the dataset.

### 2. Top 10 Most Frequent Tags
Displays the most commonly used quote tags.

### 3. Quote Length Distribution
Analyzes the distribution of quote lengths.

### 4. Quote Length Outlier Analysis
Identifies unusually short or long quotes using a box plot.

### 5. Top 5 Tags Share
Shows the percentage contribution of the most common tags using a pie chart.

## 🔍 Key Insights
- Certain authors contribute significantly more quotes than others.
- Inspirational and life-related tags appear frequently.
- Most quotes fall within a similar length range.
- A few quotes are significantly longer than average.
